# 🛡️ SecurifyAI

SecurifyAI is an AI-powered cybersecurity threat analysis platform that helps identify security threats from raw logs, assess severity, explain business impact, and recommend secure remediation steps.

## Features

- AI-Powered Threat Detection
- SQL Injection Detection
- XSS Detection
- Path Traversal Detection
- Security Score Generation
- Executive & Developer Views
- Secure Patch Recommendations
- Scan History Dashboard
- Executive Report Generation
- SQLite Database Storage
